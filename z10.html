Pytania dotyczące budowy selektora

  1. Wymienić oraz opisz dwie części selektora

    2. Selektor składa się z dwóch części:

    • Element selektora: określa rodzaj elementu HTML, na którym ma być zastosowany styl.
    • Klasy, identyfikatory, atrybuty lub położenie: dodatkowe atrybuty, które określają, na które elementy HTML ma być zastosowany styl.
  2. Znaczenie nawisów klamrowych

    3. Nawiasy klamrowe {} zawierają deklaracje właściwości CSS i ich wartości dla wybranych elementów, które zostały wybrane za pomocą selektora.

  3. Opisz cechy

    4. Cechy to właściwości CSS, które określają wygląd i zachowanie elementów HTML. Przykładowe cechy to color (kolor tekstu), font-size (rozmiar czcionki), border (ramka) itp.

  4. Opisz wartości

    5. Wartości to konkretne parametry, które nadajemy cechom CSS. Na przykład, dla cechy color wartość może być nazwą koloru (np. red), kodem koloru w formie szesnastkowej (np. #FF0000), lub funkcją RGB (np. rgb(255, 0, 0)).

  5. Kiedy stosujemy znak średnika (;), kiedy nie jest konieczny

    6. Znak średnika (;) stosujemy po każdej deklaracji właściwości CSS wewnątrz nawiasów klamrowych {}. Jest to konieczne, aby oddzielić deklaracje od siebie. Jednakże, jeśli mamy tylko jedną deklarację, nie jest konieczny znak średnika na końcu.
