Cechy modyfikowania formatowania w HTML i CSS
HTML CSS
Modyfikacja wyglądu elementów strony jest bardzo trudna. Znaczniki i atrybuty które za to odpowiadają są porozrzucane w różnych miejscach kodu i mieszają się ze strukturą kodu). Osobne przedstawienie i struktura dokumentów czyli: polecenia dotyczące formatowania można umieścić w jednym miejscu i powiązać je z konkretnymi elementami wstawionymi za pomocą czystego HTML. Taki sposób powoduję że modyfikacja wyglądu stron może przebiegać dużo szybciej.
